A **data structure** is a way of **organizing, storing, and managing data** so that it can be used efficiently.

In simple terms — it’s like a container that helps you store data in a specific format to make operations like searching, sorting, inserting, or deleting faster and easier.

---

### 🔹 Definition:

> A **data structure** is a specialized format for organizing, processing, retrieving, and storing data.

---

### 🔹 Why It’s Important:

- Helps manage large amounts of data efficiently.
    
- Improves performance of algorithms.
    
- Enables easy access and modification of data.
    
- Makes programs cleaner, faster, and easier to maintain.
    

---

### 🔹 Types of Data Structures:

#### **1. Primitive Data Structures**

These are the basic data types provided by a programming language.  
Examples:

- Integer (`int`)
    
- Float (`float`)
    
- Character (`char`)
    
- Boolean (`bool`)
    

#### **2. Non-Primitive Data Structures**

These are more complex structures built using primitive data types.

**a. Linear Data Structures** – Elements are arranged sequentially.

- **Array**
    
- **Linked List**
    
- **Stack**
    
- **Queue**
    

**b. Non-Linear Data Structures** – Elements are connected hierarchically or through relationships.

- **Tree**
    
- **Graph**
    

**c. Hash-Based Structures**

- **Hash Table / Hash Map**
    

---

### 🔹 Example:

If you want to store names of students:

- Using an **array** → You can store them in a list like `[“Ali”, “Rafi”, “Nila”]`.
    
- Using a **linked list** → Each name is a node connected to the next one.
    
- Using a **tree** → You can store students based on hierarchy (e.g., class → section → student).
