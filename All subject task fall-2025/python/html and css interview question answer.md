
## 🧱 **Section 1: HTML Basics (1–20)**

1. **What is HTML?**  
    → HyperText Markup Language, used to create web pages.
    
2. **What does HTML stand for?**  
    → HyperText Markup Language.
    
3. **What is the latest version of HTML?**  
    → HTML5.
    
4. **What is the purpose of HTML?**  
    → It structures content on the web.
    
5. **What are HTML tags?**  
    → Markup enclosed in `< >` used to define elements.
    
6. **What is an HTML element?**  
    → It consists of a start tag, content, and an end tag.
    
7. **What is the function of `<!DOCTYPE html>`?**  
    → Declares the document as HTML5.
    
8. **What is the root element in HTML?**  
    → `<html>` tag.
    
9. **What is inside the `<head>` tag?**  
    → Metadata, links, title, and scripts.
    
10. **What is inside the `<body>` tag?**  
    → The visible page content.
    
11. **What are block-level elements?**  
    → Elements that start on a new line (e.g., `<div>`, `<p>`, `<h1>`).
    
12. **What are inline elements?**  
    → Elements that do not start on a new line (e.g., `<span>`, `<a>`).
    
13. **What is the difference between `<div>` and `<span>`?**  
    → `<div>` is block-level; `<span>` is inline.
    
14. **What is the purpose of `<title>`?**  
    → Displays the page title in the browser tab.
    
15. **What are empty elements?**  
    → Elements with no closing tag (e.g., `<br>`, `<img>`).
    
16. **What is an attribute in HTML?**  
    → Provides extra information about an element.
    
17. **Where do we write attributes?**  
    → Inside the opening tag (e.g., `<a href="#">`).
    
18. **What is the use of comments in HTML?**  
    → To write notes ignored by the browser (`<!-- comment -->`).
    
19. **Is HTML case sensitive?**  
    → No, HTML is not case-sensitive.
    
20. **Can we use multiple `<h1>` tags on one page?**  
    → Yes, but it’s better to have one main `<h1>` per page for SEO.

21. **What is a semantic element?**

22. **Name some semantic elements.**

25. **What is the purpose of `<nav>`?**

26. **What is `<article>` used for?**

27. **What does `<section>` do?**

28. **What is the difference between `<section>` and `<div>`?**

29. **What does `<main>` represent?**

30. **What is the use of `<aside>`?**

31. **What does `<figure>` and `<figcaption>` do?**

32. **What is the `<iframe>` tag used for?**

36. **What is `<pre>` tag?**  
    → Displays preformatted text with spaces and line breaks preserved.
    
37. **What is `<code>` tag used for?**

40. **What is `<time>` used for?**

41. **How do you add an image in HTML?**  
    → `<img src="image.jpg" alt="Description">`
    
42. **What is the `alt` attribute used for?**  
    → Displays alternate text if image fails to load.
    
43. **What is `<a>` used for?**  
    → To create hyperlinks.
    
44. **What are absolute and relative links?**  
    → Absolute = full URL; Relative = local path.
    
45. **What is the use of `target="_blank"`?**  
    → Opens a link in a new tab.
    
46. **How to link an email in HTML?**  
    → `<a href="mailto:example@email.com">Email Us</a>`
    
47. **How to link a phone number?**  
    → `<a href="tel:+880123456789">Call Us</a>`
    
48. **How to add a background image?**  
    → Using CSS: `style="background-image: url('bg.jpg');"`

    → `<audio controls><source src="sound.mp3" type="audio/mpeg"></audio>`
    
51. **What is the purpose of the `<source>` tag?**  
    → Defines multiple media sources for `<video>` or `<audio>`.
    
52. **What is `<embed>` used for?**  
    → Embeds external content like Flash or PDF.
    
53. **What is `<object>` used for?**  
    → Embeds multimedia (alternative to `<embed>`).
    
54. **What is `<canvas>` used for?**  
    → Used for drawing graphics with JavaScript.
    
55. **What is `<svg>` used for?**  
    → To draw vector graphics.
    
56. **Can you add captions to videos?**  
    → Yes, using `<track>` tag.
    
57. **What is the `controls` attribute in `<video>`?**  
    → Displays play/pause buttons.
    
58. **What is the `autoplay` attribute?**  
    → Automatically starts media playback.
    
59. **What is the `loop` attribute?**  
    → Repeats the media automatically.
    
60. **What is the `poster` attribute in `<video>`?**  
    → Displays an image before the video plays.

## 🎯 **SECTION 1: CSS BASICS (1–20)**

1. **What does CSS stand for?**  
    → Cascading Style Sheets.
    
2. **What is CSS used for?**  
    → To style and layout web pages (e.g., colors, fonts, spacing).
    
3. **What are the three ways to apply CSS?**  
    → Inline, Internal, and External.
    
4. **What is an inline CSS?**  
    → CSS applied directly inside an element.  
    Example: `<p style="color:red;">Text</p>`
    
5. **What is internal CSS?**  
    → Written inside `<style>` tag within `<head>`.
    
6. **What is external CSS?**  
    → Written in a separate `.css` file and linked with `<link>` tag.
    
7. **What is the syntax of CSS?**
    
    ```css
    selector {
      property: value;
    }
    ```
    
8. **What is a selector?**  
    → It selects HTML elements to style.
    
9. **What are properties in CSS?**  
    → Style characteristics (e.g., color, font-size, margin).
    
10. **What are values in CSS?**  
    → Values define the property (e.g., `red`, `16px`).
    
11. **What is the difference between `id` and `class` selectors?**  
    → `id` is unique (`#id`), `class` can be reused (`.class`).
    
12. **What are universal selectors?**  
    → The `*` selector styles all elements.
    
13. **What is the purpose of grouping selectors?**  
    → To apply the same style to multiple elements.  
    Example: `h1, h2, h3 { color: blue; }`
    
14. **What is inheritance in CSS?**  
    → Some properties (like color, font) are passed from parent to child elements.
    
15. **What is the “Cascading” in CSS?**  
    → CSS applies rules in a specific priority order.
    
16. **What is specificity in CSS?**  
    → A ranking system that determines which style takes priority.
    
17. **Order of CSS specificity:**  
    → Inline > ID > Class > Element.
    
18. **What is the difference between `rem` and `em`?**  
    → `em` is relative to parent font-size, `rem` to root font-size.
    
19. **What is the default position of elements in CSS?**  
    → `static`.
    
20. **How do you add comments in CSS?**  
    → `/* This is a comment */`

57. **What does `flex-direction` do?**

## 🧩 **SECTION 4: POSITIONING & RESPONSIVE DESIGN (61–80)**

61. **What are the different CSS position values?**  
    → static, relative, absolute, fixed, sticky.
    
62. **What is `position: relative;`?**  
    → Positions element relative to its normal position.
    
63. **What is `position: absolute;`?**  
    → Positioned relative to nearest positioned ancestor.
    
64. **What is `position: fixed;`?**  
    → Fixed relative to viewport.
    
65. **What is `position: sticky;`?**  
    → Sticks to position when scrolling within a container.
    
66. **How to move an element 20px from the top?**  
    → `top: 20px; position: relative;`
    
67. **What is the difference between `min-width` and `max-width`?**  
    → Minimum and maximum width limits for responsiveness.
    
68. **What is a media query?**  
    → Used to apply styles for specific screen sizes.  
    Example:
    
    ```css
    @media (max-width: 600px) {
      body { background: lightblue; }
    }
    ```
    
69. **What is a breakpoint?**  
    → The screen width at which layout changes.
    
70. **How to make an image responsive?**  
    → `max-width: 100%; height: auto;`
    
71. **What is `vh` and `vw`?**  
    → Viewport height and width units.
    
72. **What is the use of `calc()`?**  
    → Performs calculations in CSS.  
    Example: `width: calc(100% - 50px);`
    
73. **What is the `clip-path` property?**  
    → Clips an element into a specific shape.
    
74. **What is `object-fit` used for?**  
    → Defines how images or videos fit within containers.
    
75. **What is a pseudo-class?**  
    → Styles specific states (e.g., `:hover`, `:focus`).
    
76. **What is a pseudo-element?**  
    → Styles parts of an element (e.g., `::before`, `::after`).
    
77. **Difference between pseudo-class and pseudo-element?**  
    → `:` for state; `::` for element parts.
    
78. **What does `cursor: pointer;` do?**  
    → Changes cursor to hand on hover.
    
79. **What is `opacity` in CSS?**  
    → Controls transparency (`opacity: 0.5;`).
    
80. **How to make a sticky navigation bar?**  
    → `position: sticky; top: 0;`

## ⚡ **SECTION 5: ADVANCED CSS (81–100)**

81. **What is a CSS variable?**  
    → Custom property for reusing values.  
    Example:
    
    ```css
    :root { --main-color: blue; }
    h1 { color: var(--main-color); }
    ```
    
82. **What is the difference between `transition` and `animation`?**  
    → Transition = one-time change; Animation = continuous motion.
    
83. **How to add a transition effect?**
    
    ```css
    button {
      transition: background 0.3s ease;
    }
    ```
    
84. **What is `@keyframes` used for?**  
    → Defines animation steps.
    
85. **Example of CSS animation:**
    
    ```css
    @keyframes move {
      from { left: 0; }
      to { left: 100px; }
    }
    ```
    
86. **What is a CSS preprocessor?**  
    → Tool like SASS or LESS that extends CSS features.
    
87. **What is the advantage of SASS?**  
    → Variables, nesting, mixins, and reusable code.
    
88. **What is the difference between relative and absolute units?**  
    → Relative units (`em`, `%`) depend on parent; absolute (`px`) do not.
    
89. **What is `transform` property?**  
    → Used to rotate, scale, or move elements.
    
90. **Example of rotate:**  
    → `transform: rotate(45deg);`
    
91. **What does `translate()` do?**  
    → Moves an element.  
    Example: `transform: translate(50px, 20px);`
    
92. **What is `scale()`?**  
    → Resizes an element.
    
93. **What is `skew()`?**  
    → Tilts an element.
    
94. **What is `perspective` used for?**  
    → Creates 3D depth effect.
    
95. **What is the difference between `nth-child()` and `nth-of-type()`?**  
    → `nth-child` = based on position; `nth-of-type` = based on element type.
    
96. **What is the use of `will-change` property?**  
    → Optimizes performance for upcoming animations.
    
97. **What is `filter` in CSS?**  
    → Applies effects like blur, brightness, grayscale.  
    Example: `filter: blur(5px);`
    
98. **What is a CSS reset?**  
    → Removes browser default styles.
    
99. **What is the difference between `visibility: hidden` and `opacity: 0`?**  
    → Hidden removes interactivity; opacity keeps it clickable.
    
100. **What is the difference between CSS Grid and Flexbox?**  
    → Grid = 2D layout (rows + columns); Flexbox = 1D layout (row or column).
