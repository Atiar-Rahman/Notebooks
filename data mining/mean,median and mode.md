# 🔹 **Measures of Central Tendency**

These are methods to describe the "center" of a dataset.

---

### 1. **Mean (Average)**

- **Definition**: Sum of all values ÷ Number of values.
    
- **Formula**:
    
    $\text{Mean} = \frac{\sum X}{N}$
    
    where X = values, N = total number of values.
    
- **Example**:  
    Data = {2, 4, 6, 8, 10}  
    Mean = (2+4+6+8+10)/5 = 30/5 = **6**
    

---

### 2. **Median (Middle Value)**

- **Definition**: The middle value when data is arranged in ascending order.
    
- If N is odd → middle value.
    
- If N is even → average of two middle values.
    
- **Example**:  
    Data = {2, 4, 6, 8, 10} (already sorted)  
    Median = middle = **6**  
    Data = {2, 4, 6, 8}  
    Median = (4+6)/2 = **5**
    

---

### 3. **Mode (Most Frequent Value)**

- **Definition**: The value that occurs most often in the dataset.
    
- A dataset can have:
    
    - No mode (if no repetition).
        
    - One mode (unimodal).
        
    - More than one mode (bimodal/multimodal).
        
- **Example**:  
    Data = {2, 4, 4, 6, 8, 10}  
    Mode = **4** (appears most often).
    

---

# ✅ **Quick Summary**

- **Mean** → Arithmetic average (sensitive to outliers).
    
- **Median** → Middle value (better when outliers exist).
    
- **Mode** → Most frequent value (good for categorical data).
